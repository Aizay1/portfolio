This project involves the full design and simulation of a structured network rack setup using Cisco Packet Tracer. It demonstrates the practical implementation of core networking concepts such as subnetting, IP addressing, VLAN configuration, routing (static and dynamic), and switch/router integration.

Pharmacy Management System

This is a simple yet powerful Pharmacy Management System designed to help manage essential day-to-day pharmacy operations. Built using Python with Tkinter for the GUI and SQLite for the database, this application allows users to handle medicine inventories, customer details, and sales records efficiently.

My Recent Works

Networking Rack Design & Implementation (Cisco Packet Tracer)

This project involves the full design and simulation of a structured network rack setup using Cisco Packet Tracer. It demonstrates the practical implementation of core networking concepts such as subnetting, IP addressing, VLAN configuration, routing (static and dynamic), and switch/router integration. The virtual rack includes multiple devices—routers, switches, PCs, and servers—strategically organized to reflect a real-world network topology. The design emphasizes scalability, security, and efficient data flow, making it suitable for a small to medium-sized enterprise environment. This project showcases hands-on skills in logical network planning, device configuration, and troubleshooting using Cisco best practices.

pharmacy management system

This is a simple yet powerful Pharmacy Management System designed to help manage essential day-to-day pharmacy operations. Built using Python with Tkinter for the GUI and SQLite for the database, this application allows users to handle medicine inventories, customer details, and sales records efficiently. Key features include adding and updating medicine stock, processing sales, calculating totals with automatic receipts, and maintaining a searchable database of transactions. The system is user-friendly, responsive, and ideal for small to mid-sized pharmacy setups looking for a lightweight digital solution.

Clinic Database System

I designed and implemented a Clinic Database System using MySQL to manage core operations like patient records, doctor assignments, appointment scheduling, and medical history tracking. The system was built with a fully normalized relational database structure, including foreign key constraints and optimized queries for reliability and performance. I also integrated the backend with a Python- and PHP-based interface, allowing seamless interaction with the database. This project reflects my ability to translate real-world healthcare requirements into an efficient, secure, and scalable data solution.
